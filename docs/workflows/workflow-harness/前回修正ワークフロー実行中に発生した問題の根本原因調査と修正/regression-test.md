### 変更内容

definitions.ts への追加内容は、testing フェーズと regression_test フェーズのサブエージェントテンプレートに対する以下の 2 つのガイダンス強化です。

FR-9 では、testing フェーズテンプレートに sessionToken の正しい取得経路と使用方法に関する説明を追加しました。sessionToken が Orchestrator からプロンプト引数として渡される値であること、および sessionToken を受け取らなかった場合は引数を省略して呼び出すことの明示化です。

FR-10 では、regression_test フェーズテンプレートに `workflow_record_test_result` の `output` パラメータに関する詳細要件を追加しました。生のテストフレームワーク標準出力（100 文字以上）をそのまま貼り付けることの重要性を強調し、加工・編集・要約した出力はバリデーターで検出されてエラーになることを明記しました。

definitions-subagent-template.test.ts への追加は、上記 FR-9・FR-10 に対応するテストケース 5 件です。
