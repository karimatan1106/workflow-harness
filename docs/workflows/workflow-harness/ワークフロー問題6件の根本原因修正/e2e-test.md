1. **パターン1**: 標準的なvitest出力

   - 期待: outputから件数抽出 → passedCount=5, failedCount=0

2. **パターン2**: 非標準的な出力 + summaryフォールバック
   - 入力:
     - output: 「テスト実行完了」（キーワード不足、件数抽出失敗）
     - summary: 「5件のテスト実行完了」
   - 期待: summaryから件数抽出 → passedCount=5, failedCount=0

3. **パターン3**: 非標準的な出力 + summary空文字列
   - 入力:
     - output: 「テスト実行完了」
     - summary: 「」（空文字列）
   - 期待: passedCount=0, failedCount=0で記録（フォールバック失敗時のデフォルト）

#### 実装コード確認

**コード引用: record-test-result.ts (線225-249)**
```typescript
function extractTestCountFromSummary(summary: string): { total: number } | null {
  const patterns = [
    /(\d+)件のテスト/,           // 日本語形式1: "5件のテスト"
    /(\d+)テスト実行/,           // 日本語形式2: "5テスト実行"
    /totalTests:\s*(\d+)/,       // 英語形式1: "totalTests: 5"
    /(\d+)\s+tests?/i,           // 英語形式2: "5 tests" or "1 test"
  ];

  for (const pattern of patterns) {
    const match = summary.match(pattern);
    if (match && match[1]) {
      const total = parseInt(match[1], 10);
      if (!isNaN(total) && total >= 0) {
        return { total };
      }
    }
  }

  return null;
}
```

**フォールバック処理の実装確認: record-test-result.ts (線381-392)**
```typescript
// 変更F: summaryフィールドによるフォールバック処理
// 既存のフレームワークパターンマッチが失敗した場合のフォールバック
if (!counts.passedCount && !counts.failedCount && summary) {
  const summaryCount = extractTestCountFromSummary(summary);

  } else if (summary.length > 0) {

}
```
