## 既知の警告と注釈

テスト実行中に以下の警告が出力されていますが、これらは**テスト環境固有のものであり、本番環境での動作に影響ありません**。

### Promise Rejection Handled Warning

```
(node:108172) PromiseRejectionHandledWarning: Promise rejection was handled asynchronously
```

- **原因**: vitest環境で非同期Promise拒否がテスト終了後に処理される
- **影響**: なし（テスト結果に含まれない）
- **対処**: テスト環境のPromise生存期間がテストフレームワークの制御外で発生
- **状態**: 予期されたテスト警告、リグレッション検出なし

### Mock Export Warning

```
[vitest] No "exportGlobalRules" export is defined on the "artifact-validator.js" mock
```

- **原因**: vitest のモック設定で特定エクスポートがオプショナル
- **影響**: なし（代替パスが実装されている）
- **対処**: テストスイートで代替ロジックを使用
- **状態**: 既知の非crítica警告、全テスト合格

### fs mock Warning

```
[vitest] No "mkdirSync" export is defined on the "fs" mock
```

- **原因**: ファイルシステム操作がモックされている設計
- **影響**: なし（テストはメモリ上で実行）
- **対処**: キャッシュ永続化は本番環境で実施
- **状態**: 既知の非crítica警告、ユニットテスト整合性に影響なし
